# Page Conversion Rate Optimization (CRO)

> Source: marketingskills/page-cro
> Use for: Improving conversion rates on marketing pages

## When to Use

- Page has traffic but low conversions
- Bounce rate is high
- Users aren't taking desired action
- Optimizing landing pages
- Improving pricing page conversions

## Initial Assessment

Before optimizing, identify:

1. **Page Type**: Homepage, landing page, pricing, feature, blog
2. **Primary Goal**: The ONE action visitors should take
3. **Traffic Source**: Where visitors come from (intent matters)

## 7-Dimension Analysis Framework

### 1. Value Proposition Clarity

**Question:** Can visitors understand what this is and why it matters in 5 seconds?

**Checklist:**
- [ ] Headline communicates core value
- [ ] Benefits before features
- [ ] Customer language, not jargon
- [ ] Clear differentiation

### 2. Headline Effectiveness

**Strong headline patterns:**
```
✅ "Get [outcome] without [pain point]"
✅ "[Number] teams use [Product] to [achieve X]"
✅ "[Outcome] in [timeframe]"
✅ "Stop [pain]. Start [benefit]."
```

**Weak headlines:**
```
❌ "Welcome to [Company]"
❌ "The Future of [Buzzword]"
❌ "[Company Name] Platform"
```

### 3. CTA Placement & Copy

**Requirements:**
- [ ] One clear primary CTA above fold
- [ ] CTA text communicates value ("Start Free Trial" > "Submit")
- [ ] CTAs repeated at decision points
- [ ] Visual hierarchy makes CTA obvious
- [ ] Secondary CTAs don't compete

### 4. Visual Hierarchy & Scannability

**Checklist:**
- [ ] Information flows logically
- [ ] Sufficient white space
- [ ] Most important elements visually prominent
- [ ] Page is scannable (can skim and understand)
- [ ] Consistent visual patterns

### 5. Trust Signals & Social Proof

**Placement:** Near CTAs and benefit claims

**Types:**
- Customer logos
- Testimonials with attribution
- Case studies with numbers
- Review scores (G2, Capterra)
- Security badges
- Media mentions

### 6. Objection Handling

**Common objections to address:**
- "Is this right for me?"
- "What if it doesn't work?"
- "How hard is setup?"
- "What about [competitor]?"
- "Is my data safe?"

**Methods:**
- FAQ section
- Guarantee/risk reversal
- Comparison sections
- Process transparency
- Support availability

### 7. Friction Points

**Eliminate:**
- Unnecessary form fields
- Unclear navigation
- Required info that shouldn't be
- Mobile experience issues
- Slow loading
- Confusing next steps

## Page-Specific Guidance

### Homepage

- Clear positioning for cold visitors
- Self-selection paths (by role/use case)
- Handle both ready-to-buy and researching
- Strong hero with primary CTA
- Social proof early

### Landing Pages

- Message-match with traffic source
- Single CTA (not multiple choices)
- Complete argument on one page
- Genuine urgency/scarcity if applicable
- Minimal navigation

### Pricing Pages

- Clear plan comparison
- Recommended plan indicated
- Feature clarity (what's included)
- Reduce plan-selection anxiety
- Easy path to checkout
- FAQ addressing pricing questions

### Feature Pages

- Feature → Benefit connection
- Use cases with examples
- Competitor comparison (if applicable)
- Clear CTA to try feature
- Screenshots/demos

### Blog Posts

- Contextual CTAs matching content
- Lead magnets related to topic
- Inline CTAs at natural stops
- Exit-intent backup CTA
- Related content suggestions

## Quick Wins Framework

**Immediate impact, easy to implement:**
1. Improve CTA copy (action + value)
2. Add social proof near CTAs
3. Simplify above-fold messaging
4. Add FAQ section for objections
5. Remove unnecessary form fields

## High-Impact Changes

**More effort, significant gains:**
1. Complete hero section rewrite
2. Add video demo/explanation
3. Restructure page flow
4. Create role-specific landing pages
5. Implement exit-intent strategies

## A/B Test Ideas

For each page, consider testing:
- Headline variations (3 options)
- CTA button text
- CTA button color/size
- Hero image vs. video
- Social proof placement
- Form length (short vs. long)
- Pricing display format

## Measurement

Track per page:
- Conversion rate (primary goal)
- Bounce rate
- Time on page
- Scroll depth
- CTA click rate
- Form completion rate
