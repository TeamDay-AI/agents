# SEO Copywriting

> Source: marketingskills/copywriting
> Use for: Writing titles, meta descriptions, and page copy that converts

## When to Use

- Optimizing title tags for CTR
- Writing meta descriptions
- Improving headline effectiveness
- Rewriting page copy for conversions
- Creating compelling CTAs

## Core Principles

1. **Clarity over cleverness** - Be direct, not cute
2. **Benefits over features** - What do they GET, not what it DOES
3. **Specificity wins** - Numbers, timeframes, concrete outcomes
4. **Customer language** - Use their words, not jargon

## Title Tag Optimization

### Title Formula Patterns

**Outcome-focused:**
```
[Get/Achieve] [Desired Outcome] [Without Pain Point]
Best Free AI Models 2026 | No Credit Card Required
```

**Comparison:**
```
[A] vs [B]: [Key Differentiator] [Year]
GPT-5.2 vs DeepSeek: 100x Price Difference Compared
```

**List/Number:**
```
[Number] [Things] [for Outcome] [Year]
47 Free AI Models on OpenRouter 2026 (Complete List)
```

**Question:**
```
[How/What/Why] [Problem/Goal]?
What's the Cheapest AI Model on OpenRouter?
```

**"How to":**
```
How to [Achieve Goal] [in Timeframe/Without Pain]
How to Cut AI Costs by 90% with OpenRouter
```

### Title Tag Best Practices

| Element | Recommendation |
|---------|----------------|
| Length | 50-60 characters |
| Primary keyword | Near the front |
| Numbers | Include when relevant |
| Year | Add for time-sensitive content |
| Brand | At end if space allows |
| Separators | Use `|` or `-` or `:`|

### CTR Boosters for Titles

- **Numbers**: "47 Free Models" beats "Many Free Models"
- **Brackets**: [2026] [Updated] [Free] [Complete Guide]
- **Power words**: Best, Ultimate, Complete, Free, Proven
- **Freshness**: Include year/month for dated content
- **Specificity**: "$0.25 per million" beats "cheap"

## Meta Description Optimization

### Meta Description Formula

```
[Value proposition/hook]. [Specific benefit with number]. [CTA or urgency].
```

**Example:**
```
Compare 400+ AI models on OpenRouter (Updated Jan 2026).
Find the cheapest models, best FREE options, and save up to 100x on AI costs.
```

### Meta Description Best Practices

| Element | Recommendation |
|---------|----------------|
| Length | 150-160 characters |
| Primary keyword | Include naturally |
| Call to action | Optional but effective |
| Unique value | What makes this page special |
| Match intent | Address what searcher wants |

### Meta Description Patterns

**For comparison pages:**
```
[A] vs [B] compared: [Key difference]. See [specific detail],
pricing, and which is best for [use case].
```

**For how-to content:**
```
Learn how to [achieve goal] in [timeframe]. [Specific method/tool].
[Credential or proof point].
```

**For product pages:**
```
[Product] helps you [achieve outcome] without [pain point].
[Social proof]. Try free.
```

## H1 Optimization

### H1 Best Practices

- **One H1 per page** (SEO standard)
- **Match title tag intent** (don't confuse Google)
- **Slightly different wording** is fine
- **Front-load keywords** when natural
- **Avoid generic H1s** like "Welcome" or "Home"

### H1/Title Alignment

```
Title: Best Free AI Models on OpenRouter 2026 | No Credit Card | Llama, Gemini
H1: Best Free AI Models on OpenRouter 2026 – No Credit Card Required
```

Both target same intent, slightly different formatting.

## CTA Copy

### CTA Formula

```
[Action Verb] + [What They Get]
```

**Examples:**
```
✅ Start Free Trial
✅ Get the Complete Guide
✅ Compare Models Now
✅ Download Free Template

❌ Submit
❌ Click Here
❌ Learn More (too vague)
```

### CTA Placement

- **Above the fold** - Primary CTA visible without scrolling
- **After value propositions** - When benefits are fresh
- **End of sections** - Natural decision points
- **Before exit** - Footer or exit-intent

## Writing Style Rules

### Do This

- **Simple words**: Use "use" not "utilize"
- **Active voice**: "You can save" not "Savings can be achieved"
- **Second person**: "You" and "Your"
- **Short paragraphs**: 2-3 sentences max
- **Bullet points**: For scanning
- **Specific claims**: "$0.25/M tokens" not "affordable"

### Avoid This

- **Jargon**: Unless your audience uses it
- **Filler words**: "In order to", "It is important to note"
- **Passive voice**: "The feature was added" → "We added"
- **Vague superlatives**: "Best", "Amazing", "Revolutionary"
- **Wall of text**: Break up long paragraphs

## Content Structure for SEO

### Above the Fold

1. **H1** - Clear, keyword-rich
2. **Subheadline** - Expand on H1 with benefit
3. **Primary CTA** - Clear next step
4. **Trust signal** - Logo bar, rating, or proof point

### Body Structure

```markdown
## [Benefit-focused H2]
[Opening that addresses reader need]
[Evidence/explanation]
[CTA or transition]

## [Next H2 - answer common question]
...
```

### FAQ Sections

Add FAQ section to pages for:
- Long-tail keyword coverage
- Featured snippet potential
- FAQ schema opportunity
- User intent satisfaction

## Output Format for SEO Copy

When optimizing a page, provide:

```markdown
## Title Tag
Current: [existing title]
Recommended: [new title] (X chars)
Rationale: [why this is better]

## Meta Description
Current: [existing]
Recommended: [new] (X chars)
Rationale: [why]

## H1
Current: [existing]
Recommended: [new]

## Alternative Headlines (for A/B testing)
1. [Option A]
2. [Option B]
3. [Option C]
```
