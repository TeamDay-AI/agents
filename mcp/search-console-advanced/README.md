# Search Console Advanced MCP Server

Feature-rich Google Search Console server by [AminForou](https://github.com/AminForou). 18 tools covering analytics, URL inspection, sitemap management, and period comparisons.

- **Source**: [AminForou/mcp-gsc](https://github.com/AminForou/mcp-gsc)
- **Install**: Requires git clone + Python 3.11+ venv + `pip install -r requirements.txt`
- **Auth**: OAuth (recommended) or service account
- **Tools**: 18

## Comparison with search-console-mcp

| Feature | search-console-mcp (ours) | search-console-advanced |
|---|---|---|
| Language | Node.js | Python 3.11+ |
| Tools | 5 | 18 |
| URL inspection | No | Yes (single + batch) |
| Sitemap management | No | Full CRUD |
| Period comparison | No | Yes |
| Install friction | Low | High (venv required) |

Use **search-console-mcp** for quick, lightweight analytics. Use **search-console-advanced** when you need URL inspection, sitemap management, or period comparisons.
