# Google Ads MCP Server

Official Google Ads MCP server by Google. Query ad accounts using GAQL and list accessible customers. Experimental status.

- **Source**: [googleads/google-ads-mcp](https://github.com/googleads/google-ads-mcp)
- **Install**: `pipx run --spec git+https://github.com/googleads/google-ads-mcp.git google-ads-mcp`
- **Auth**: Google ADC + Ads Developer Token
- **Tools**: 2 (search, list_accessible_customers)
- **Note**: Requires a Google Ads developer token from your Google Ads account
