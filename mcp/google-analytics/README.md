# Google Analytics MCP Server

Official Google Analytics 4 MCP server by Google. Query accounts, run standard and real-time reports, and access custom dimensions.

- **Source**: [googleanalytics/google-analytics-mcp](https://github.com/googleanalytics/google-analytics-mcp)
- **Install**: `pipx run analytics-mcp`
- **Auth**: Google ADC (service account or OAuth)
- **Tools**: 6 (accounts, reports, real-time, custom fields)
