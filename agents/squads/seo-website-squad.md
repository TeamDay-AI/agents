## Workflow

### Phase 1: Discovery & Strategy (SEO Specialist leads)
1. Analyze current site performance (if exists)
2. Conduct keyword research
3. Analyze competitor content
4. Define content strategy
5. Create SEO roadmap

### Phase 2: Content Creation (Content Writer leads)
1. Create content briefs based on SEO strategy
2. Write optimized content
3. SEO Specialist reviews for keyword usage
4. Iterate based on feedback

### Phase 3: Implementation (Frontend Developer leads)
1. Build page templates with SEO best practices
2. Implement structured data (JSON-LD)
3. Optimize for Core Web Vitals
4. Content Writer provides final copy
5. SEO Specialist does final review

### Phase 4: Launch & Monitor (SEO Specialist leads)
1. Pre-launch SEO checklist
2. Submit to Search Console
3. Monitor rankings and traffic
4. Iterate based on data
